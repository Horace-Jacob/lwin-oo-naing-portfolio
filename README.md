# Lwin Oo Naing Portfolio (2024)

## Technologies

- ReactJs
- TypeScript
- SCSS
- GSAP 3
- ThreeJs

## ** How to run the project **

1. Clone the project repository

```bash
git clone https://github.com/Lwin-Oo-Naing/portfolio.git
```

2. Install dependencies

```bash
yarn
```

3. Start the project

```bash
yarn start
```
